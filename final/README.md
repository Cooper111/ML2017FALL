# Delta Chinese QA

## Run testing
  
  1. Put **test-v1.1.json** in **src/R-net/data/**

  2. Run src/R-net/test.sh using bash (**bash test.sh**)
  
  3. The result will be **src/R-net/prediction.csv**

  pytorch 0.3.0
