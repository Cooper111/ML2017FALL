# Delta Chinese QA

## Run testing
  
  1. **cd src/R-net**
  
  2. Put **test-v1.1.json** in **data/**

  3. Run test.sh using bash (**bash test.sh**)
  
  4. The result will be **prediction.csv**

  pytorch 0.3.0

  gensim 3.1.0

  six 1.11.0

  tqdm 4.19.4
